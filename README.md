# Personal Expense Tracker (Java)

This is a simple console-based Personal Expense Tracker application developed using Core Java.  
It helps users add, view, update, delete, and calculate their daily or monthly expenses.

## Features
- Add new expenses
- View all expenses
- Update an existing expense
- Delete an expense
- Calculate total expenses
- Auto-generate Expense ID
- Uses OOP concepts like:
  - Encapsulation
  - Inheritance
  - Abstraction
  - Polymorphism

## Technologies Used
- Java
- OOP Concepts
- Scanner for input
- ArrayList for data storage

## How to Run
1. Download or clone this repository  
2. Open the project in any Java-supported IDE (VS Code, IntelliJ, Eclipse)  
3. Compile and run the `MainApp.java` file  
4. Use the menu to interact with the application
